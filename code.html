The tables below don't include all HTML tags,
but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
div division defines a section of the document
header header container for introductory content or a set of navigational links
ol ordered list defines an ordered list (numerical or alphabetical)
ul unordered list defines an unordered (bulleted) list
blockquote blockqoute specifies a section that is quoted from another source
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
span span select inline area for CSS styling <span>make it really pop with CSS</span> make it really pop with CSS
li list item defines a list item <li>Pick up milk</li>
  1. Pick up milk
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="https://andreysyroyezhkin.com/">My Website</a> My Website
img image include an image <img src="http://bit.ly/ruby-logo" alt="Ruby Logo">
code code code text formatting <code><em>emphasize</em></code> <em>emphasize</em>

Document Tags

Tag Name Purpose
html HTML tells the browser that this is an HTML document
head head container for all the head elements (title, scripts, meta)
body body defines the document's body (text, hyperlinks, images, tables, lists, etc.)
title title required in all HTML documents and it defines the title of the document
meta meta provides metadata about the HTML document. Metadata will not be displayed on the page, but will be machine parsable.
style style defines style information for an HTML document
